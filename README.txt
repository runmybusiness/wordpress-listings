== Description ==

This plugin allows you to display your RunMyBusiness property listings on your own website.

== Installation ==
 - Install the plugin
 - Set your API Keys
 - Set your listings url
 - Set your team members url
 - Run the manual sync and/or setup your wordpress cron job to run every hour
